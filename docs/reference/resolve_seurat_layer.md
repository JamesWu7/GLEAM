# Resolve Seurat assay/layer with v4/v5 compatibility

Resolve Seurat assay/layer with v4/v5 compatibility

## Usage

``` r
resolve_seurat_layer(object, assay = NULL, layer = NULL, slot = NULL)
```

## Arguments

- object:

  Seurat object.

- assay:

  Assay name.

- layer:

  Layer name.

- slot:

  Legacy slot name.

## Value

List with assay and layer/slot.

# Z-mean pathway scoring on matrix input

Z-mean pathway scoring on matrix input

## Usage

``` r
score_zmean_matrix(expr, genesets, verbose = TRUE)
```

## Arguments

- expr:

  Gene-by-cell matrix.

- genesets:

  Named list of pathways.

- verbose:

  Whether to print progress.

## Value

Pathway-by-cell numeric matrix.

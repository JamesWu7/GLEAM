# Get palette colors

Get palette colors

## Usage

``` r
get_palette(
  name = "gleam_discrete",
  n = 8,
  continuous = FALSE,
  reverse = FALSE
)
```

## Arguments

- name:

  Palette name.

- n:

  Number of colors.

- continuous:

  Whether to return continuous palette function output.

- reverse:

  Reverse color order.

## Value

Character color vector.

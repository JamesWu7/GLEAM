# Plot spatial comparison result

Plot spatial comparison result

## Usage

``` r
plot_spatial_compare(x, top_n = 20, alpha = 0.85, theme_params = list())
```

## Arguments

- x:

  `gleam_test` object or result table.

- top_n:

  Number of signatures to display.

- alpha:

  Bar alpha.

- theme_params:

  Optional list passed to
  [`gleam_theme()`](https://JamesWu7.github.io/GLEAM/reference/gleam_theme.md).

## Value

A `ggplot` object.

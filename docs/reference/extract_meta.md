# Extract metadata from input

Extract metadata from input

## Usage

``` r
extract_meta(object = NULL, meta = NULL, expr = NULL, seurat = TRUE)
```

## Arguments

- object:

  Seurat object.

- meta:

  Metadata data.frame for matrix mode.

- expr:

  Expression matrix for matrix mode.

- seurat:

  Input mode.

## Value

Metadata data.frame aligned to cells.

# GLEAM: Gene-set and cell-state exploration across space and time in R

Matrix-native signature and cell-state analysis toolkit for single-cell
and spatial transcriptomics, with optional Seurat v4/v5, trajectory, and
ecosystem integrations. Use `citation("GLEAM")` for citation metadata.

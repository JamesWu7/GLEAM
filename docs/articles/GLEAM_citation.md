# How to cite GLEAM

## Recommended citation

``` r

citation("GLEAM")
```

Copy-ready format:

Woo J (2026). **GLEAM: Gene-set and cell-state exploration across space
and time in R**. R package version 0.2.0.
<https://github.com/JamesWu7/GLEAM>

## Citation file

GLEAM provides an `inst/CITATION` file so standard R citation workflows
(`citation("GLEAM")`) resolve package citation metadata consistently.

## Methods citation notes

If you use optional backends (e.g., `msigdbr`, `UCell`, `AUCell`,
`GSVA`, `singscore`, `monocle`, `monocle3`, `slingshot`, `tradeSeq`),
cite those tools in addition to GLEAM.
